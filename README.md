<H1>Book Management Application</H1>
Welcome to the Book Management Application! This application allows users to search for books, add new books, and edit book details. The app features a responsive design, providing functionalities like pagination, filtering, and sorting.

<ul>Features :
<li>Search Books: Quickly search for books by their title.</li>
<li>Filter Books: Filter the book list by language to find specific books.</li>
<li>Add Books: Add new books to the database with a simple form.</li>
<li>Edit Books: Edit existing book details to keep the information up-to-date.</li>
<li>Responsive Design: Enjoy a seamless experience on both desktop and mobile devices.</li>
</ul>
<ul>
  Technologies Used :
  <li>React JS: A JavaScript library for building user interfaces.</li>
  <li>Axios / Fetch: This is used to make HTTP requests to the API.</li>
  <li>Redux: A state management library for managing the app's state.</li>
  <li>HTML: The standard markup language for creating web pages.</li>
  <li>CSS: Used to describe the presentation of web pages.</li>
</ul>
